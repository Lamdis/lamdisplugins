FAQ

Comment installer un plugin ?

Télécharge le plugin, place le fichier .jar dans le dossier plugins de ton serveur Minecraft, puis redémarre le serveur.

Les plugins sont-ils gratuits ?

Oui, tous les plugins listés ici sont gratuits et libres d'utilisation !

Sur quelles versions de Minecraft fonctionnent les plugins ?

Chaque plugin indique la version de Minecraft compatible dans sa fiche.

Je veux proposer mon plugin, comment faire ?

Contacte-moi via la page Contact ou ouvre une issue sur GitHub.
